Создать Git проект и загрузить в репозиторий.

1) Методология БЭМ ✓
2) Предпроцессор SASS ✓
3) Сборщик frontend (grunt, gulp, webpack на выбор) ✓
4) Адаптивность (от 320 до 1920 px)
5) НЕ использовать bootstrap ✓
7) Javascript
	- Fixed меню при скроле ✓
	- Открытие каталога и переключение табов ✓
	- Слайдер фото товара ✓
	- переключение при клике на тумб ✓
	- оценка товара (stars) ✓
		- Не использовать плагины ✓
		- выбор значения и отправка ajax запроса ✓
